Jump To Open Source V1.0.0
===============================

Description
-----------

>이 저장소는 오픈소스의 소개와 역사에 대해 쓰여진 웹페이지의 소스입니다.


Installation
------------

react로 만들어 졌으며 
nodejs가 필요합니다.

~~~
$ npm install

$ npm start
or
$ npm run build
~~~

빌드후 document 폴더를 static 폴더로 옮겨야합니다.
